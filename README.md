# Academic Skills

让 AI 帮你下载学术论文。告诉它论文标题、DOI 或关键词，它会自动找到并下载 PDF。

支持 arXiv、Nature、Springer、MDPI、知网等主流学术平台，覆盖中英文论文。

## 它能做什么

- **一句话下载** — "帮我下载 attention is all you need" → 自动找到 PDF 并保存
- **批量检索** — "找5篇关于大语言模型幻觉问题的最新论文" → 逐篇检索，自动下载能拿到的
- **知网论文** — 自动检索知网获取论文信息，优先从期刊官网免费获取 PDF，省去知网付费
- **智能路由** — 自动判断最优下载渠道：直接链接秒下、OA 论文走免费通道、出版商页面自动导航点击

## 两阶段工作流

### 阶段一：检索

根据你的需求自动搜索论文，收集标题、作者、DOI 等元数据。中文走知网，英文走 Google Scholar。少量结果直接展示，大量结果整理成 Excel。

### 阶段二：下载

对每篇论文逐级尝试下载：

| 层级 | 方式 | 说明 |
|------|------|------|
| Tier 1 | 直接下载 | 走 arXiv / Unpaywall / 期刊官网等，有直接 PDF 链接的秒下 |
| Tier 2 | 浏览器自动操作 | 自动打开出版商页面，模拟点击 PDF 按钮下载。Tier 2 结束后所有 OA 论文应已下载完成 |
| Tier 3 | 需登录的平台 | 知网等需要登录的平台，仅在机构账号有免费额度时自动下载，绝不主动付费 |

下不了的论文会标注原因（需付费 / 需机构权限等），附上链接方便你手动处理。

## 安装

将以下提示词复制发送给你的 AI 助手（Claude Code / Codex CLI）：

```
读这个安装手册帮我装好论文下载工具: https://raw.githubusercontent.com/kanlac/academic-skills/main/docs/installation-guide.md
```

AI 会全程引导你完成配置，大约 3 分钟。
