# Academic Skills

让 AI 帮你下载学术论文。告诉它论文标题、DOI 或关键词，它会自动找到并下载 PDF。

支持 arXiv、Nature、Springer、MDPI、知网等主流学术平台，覆盖中英文论文。

## 它能做什么

- **一句话下载** — "帮我下载 attention is all you need" → 自动找到 PDF 并保存
- **批量检索** — "找5篇关于大语言模型幻觉问题的最新论文" → 逐篇下载
- **知网论文** — 自动检索知网，优先从期刊官网免费获取 PDF，省去知网付费
- **智能路由** — arXiv 秒下；OA 论文走免费通道；出版商页面自动导航点击

## 三层下载策略

| 层级 | 方式 | 说明 |
|------|------|------|
| Tier 1 | 直接下载 | 毫秒级，走 arXiv / Unpaywall / 期刊官网等免费渠道 |
| Tier 2 | 浏览器自动操作 | 打开出版商页面，自动点击 PDF 按钮下载 |
| Tier 3 | 知网登录下载 | 最后手段，需要知网账号且有下载额度 |

90% 的论文在 Tier 1 就能搞定，无需启动浏览器。

## 安装

将以下提示词复制发送给你的 AI 助手（Claude Code / Claude Desktop / Codex）：

```
读这个安装手册帮我装好论文下载工具: https://raw.githubusercontent.com/kanlac/academic-skills/main/docs/installation-guide.md
```

AI 会全程引导你完成配置，大约 3 分钟。

## 使用示例

安装完成后，直接对 AI 说：

```
下载这篇论文 https://arxiv.org/abs/2301.08745
```

```
帮我找并下载：Attention Is All You Need
```

```
下载 DOI 10.1038/s41586-024-07386-0
```

```
从知网搜索"大语言模型"相关论文并下载
```

## 工作原理

```
你的请求 → AI 提取论文信息
                │
                ├─ 有直接链接？ → 秒下 PDF（Tier 1）
                │
                ├─ 中文论文？ → 知网检索元数据 → 期刊官网免费下载
                │
                └─ 需要浏览器？ → 自动启动 Chrome → 导航 → 点击下载（Tier 2）
```

知网的价值是检索中文论文信息，但实际下载优先走期刊官网（大量中文 OA 期刊在官网免费提供 PDF，知网对同样的论文却要收费）。
